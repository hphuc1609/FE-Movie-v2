# Mephim247

Free online movie website with no ads and fast loading pages.

## Official Link

https://mephim247.vercel.app

## Technology

- Next 14
- TailwindCSS
- Shadcn
- Swiper
- Axios
- Tanstack Query
- React Hook Form + Zod

## Features

- Fully responsive
- Login / Register
- Movie by category
- Comments
- Support vietsub,...
