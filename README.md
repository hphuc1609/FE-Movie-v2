# Mephim247

Free online movie website with no ads and fast loading pages.

## Official Link

https://mephim247.vercel.app

## Technology

Frontend:

- Next 14
- Tailwind
- Shadcn/ui
- Swiper
- Axios
- Tanstack Query
- React Hook Form
- Zod

Backend:

- Nodejs
- Express
- MongoDB
- JWT

## Features

- Fully Responsive Design
- Login / Register
- Filtering
- Comment
- QR code
- Support vietsub and more
